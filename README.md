== Ordered Stack-List of People (C Implementation) ==
# Overview

This project demonstrates the importance and applications of stacks and ordered lists in the C programming language.
It implements a stack-like linked list (Lista de Pessoas) that maintains elements ordered alphabetically by name and surname, illustrating the combination of ordered data structures and stack principles (LIFO removal).

Efficient insertion in an ordered manner without losing stack-like behavior for removal.
